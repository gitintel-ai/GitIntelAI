# Blog Post Outline: "I analyzed 50 open-source repos and found X% of recent code is AI-generated"

## Goal
Viral dev blog post that provides genuinely useful data, positions GitIntel as the tool that made the analysis possible, and drives organic discovery.

---

## Methodology

### Signals Used to Detect AI-Generated Code
1. **Co-Authored-By trailers** -- `Co-Authored-By: Claude`, `Co-Authored-By: Copilot`, etc. (highest confidence)
2. **Commit message patterns** -- "Generated by Claude Code", "feat: add X" with suspiciously perfect formatting
3. **Bot committer emails** -- `github-actions[bot]`, `dependabot`, etc. (excluded -- these are automation, not AI coding)
4. **Commit cadence patterns** -- rapid sequences of commits within minutes (suggestive, not definitive)
5. **PR descriptions** -- references to AI tools, "AI-generated", "written with Claude/Cursor"

### What We're NOT Claiming
- We can only detect AI code that leaves a trail (trailers, signatures). The real number is likely higher.
- `Co-Authored-By` could be added manually for non-AI reasons (rare, but possible).
- This is a lower bound, not a census.

### Tools
- GitIntel `scan` for automated detection
- Manual verification of a sample (50 commits per repo, spot-checked)
- All data publicly verifiable -- repos are all open source

- **Web frameworks:** Next.js, Astro, SvelteKit, Remix
- **AI/ML:** LangChain, LlamaIndex, Ollama, vLLM
- **Infra/DevTools:** Turborepo, Biome, Deno, Bun
- **Databases:** Drizzle ORM, Prisma, SurrealDB

### Tier 2: Developer Tools (15 repos)
- Zed editor, Cursor (if open-source components), Warp terminal
- Shadcn/ui, Radix, TanStack
- Hono, tRPC, Effect-TS
- Tauri, Leptos, Ruff

### Tier 3: Varied Languages (10 repos)
- Rust: Polars, typst, Helix
- Go: Caddy, k3s, Charm tools
- Python: FastAPI, Pydantic, httpx, Rye

### Tier 4: AI-Native Projects (10 repos)
- Projects that started in 2025-2026 (born in the AI era)
- Claude Code itself (anthropic repos)
- OpenAI repos (openai/openai-python, etc.)
- Vercel AI SDK, AI SDK RSC

### Selection Criteria
- Must have 100+ commits in the last 6 months
- Must be actively maintained (commit in last 30 days)
- Mix of languages: TypeScript, Rust, Python, Go
- Mix of project ages: established (5+ years) vs born-in-AI-era (< 2 years)
